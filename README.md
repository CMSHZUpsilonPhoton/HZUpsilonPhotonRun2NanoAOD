# HZ --> Upsilon + Photon - Run 2 - NanoAOD


## Setup
Assuming a CC7 machine with miniconda installed.

Install miniconda:

https://docs.conda.io/en/latest/miniconda.html

```
conda create -y -c conda-forge -p env --file requirements.txt
```

## Load env

```
conda activate ./env
```
## Run

```
./run_analysis.py
```
